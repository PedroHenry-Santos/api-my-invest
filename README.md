## Description

API da aplicação [Nois Inevst](https://noisinvest.com/).

## Installation

```bash
$ git clone git@github.com:PedroHenry-Santos/api-nois-invest.git
```

## Running the app

```bash
$ chmod +x ./run.sh
$ chmod +x ./docker/api/ssh.sh
$ chmod +x ./docker/api/start.sh
$ chmod +x ./docker/postgresql/ssh.sh

$ yarn install

$ make up
```
